### Hardware

* **MOBO:** MSI Z87-GD65 GAMING
* **RAM:** G.Skill DDR3-2400 16GB (2x8GB)
* **CPU:** Intel Core i5-4690K
* **GPU:** AMD Radeon RX 580 8GB
* **SSD:** Crucial MX500 500GB
* **WIFI:** Fenvi FV-T919
